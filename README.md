# SpiritQuest

An anime-inspired, video game based habit tracker built with React Native and Expo.

## Overview

SpiritQuest is a mobile habit tracker designed to make daily consistency feel rewarding and fun. Using gamification elements like streaks, motivational nudges, milestone badges, progress graphs, and custom reminder times, it helps users build lasting habits.

Built as a senior project in September-December 2025.

## Features

- Local user registration
- Create habits with name, frequency, and custom daily reminder time
- Daily logging with streak tracking and automatic reset for missed days
- Rule-based motivational nudges (low and high streak messages using json-rules-engine)
- Badge unlocks at 7, 14, and 30 days with anime-style icons
- Per-habit line graphs and aggregated weekly summary graph
- Edit and delete habits
- Full persistence using AsyncStorage
- Cyberpunk 2077-inspired dark neon theme

## Stretch Goals Completed

- Custom reminder times per habit
- Anime-themed badge icons
- Aggregated weekly summary graph
